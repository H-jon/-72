# MOODIE

기분과 상황에 따라 영화를 추천하는 반응형 웹 포트폴리오입니다.

## 주요 기능

- 기분 기반 영화 추천
- 액션, 호러, 로맨스, 코미디, 드라마, 애니메이션, SF, 미스터리 각 20편
- 총 160편의 영화 탐색
- CSS 기반 영화 포스터와 미리보기
- 장르 필터와 제목 검색
- 영화 상세 모달
- `localStorage` 기반 찜 목록
- PC, 태블릿, 모바일 반응형 UI

## 실행

별도 설치 없이 `index.html`을 브라우저에서 열면 됩니다.

GitHub Pages를 사용하는 경우 저장소의 Pages 설정에서 `main` 브랜치 루트를 배포 대상으로 선택하세요.
