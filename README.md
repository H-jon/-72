# 🎬 MOODIE

**MOODIE는 사용자의 기분과 관람 상황을 바탕으로 어울리는 영화를 추천해 주는 영화 큐레이션 웹사이트입니다.**

오늘의 기분, 함께 보는 사람, 원하는 상영시간 등을 선택하면 조건에 맞는 영화를 추천받을 수 있습니다. 추천 결과뿐 아니라 장르별 영화 탐색, 영화 상세정보 확인, 찜 목록 저장 기능까지 한 번에 이용할 수 있도록 구성했습니다.

## 🌐 웹사이트

- 배포 페이지: https://h-jon.github.io/-72/
- GitHub 저장소: https://github.com/H-jon/-72

## ✨ 주요 기능

- 사용자의 기분을 반영한 영화 추천
- 동행인·상영시간·추천 기준 선택
- 추천 영화 결과와 미리보기 제공
- 영화 제목 및 장르 검색
- 장르별 영화 필터링
- 영화 상세정보 확인
- 관심 영화 찜하기 및 목록 관리
- 모바일·태블릿·PC 반응형 화면 지원

## 🖼️ 포스터 이미지

프로젝트에 포함된 실제 영화 포스터 30개를 로컬 WebP 이미지 스프라이트로 구성했습니다.

외부 이미지 서버에 의존하지 않기 때문에 인터넷 환경이나 외부 링크 상태와 관계없이 GitHub Pages에서도 포스터가 안정적으로 표시됩니다. 포스터 영역은 2:3 비율로 유지되며, 카드 크기에 맞춰 자연스럽게 출력됩니다.

## 🛠️ 사용 기술

- **HTML5** — 웹페이지 구조
- **CSS3** — 레이아웃, 포스터 카드, 반응형 디자인
- **JavaScript** — 영화 추천, 검색, 필터, 상세정보 및 찜 기능
- **LocalStorage** — 사용자가 선택한 조건과 찜 목록 저장
- **GitHub Pages** — 정적 웹사이트 배포

## 📄 페이지 구성

| 파일 | 설명 |
|---|---|
| `index.html` | MOODIE 소개 및 메인 화면 |
| `mood.html` | 현재 기분 선택 |
| `conditions.html` | 동행인·상영시간·추천 기준 선택 |
| `results.html` | 추천 결과, 검색 및 영화 탐색 |
| `detail.html` | 선택한 영화의 상세정보 |
| `favorites.html` | 사용자가 찜한 영화 목록 |

## 🚀 실행 방법

1. 프로젝트 ZIP 파일의 압축을 풉니다.
2. 압축을 푼 폴더에서 `index.html`을 실행합니다.
3. 별도의 설치나 서버 설정 없이 브라우저에서 바로 사용할 수 있습니다.

기존 파일 위에 덮어쓴 경우 브라우저 캐시로 이전 화면이 보일 수 있습니다. 이때는 **Ctrl + F5**를 눌러 강력 새로고침하세요.

## 📌 프로젝트 목적

이 프로젝트는 HTML, CSS, JavaScript만으로 영화 추천 서비스의 전체 사용자 흐름을 구현한 웹 포트폴리오입니다. 사용자가 복잡한 설정 없이 직관적으로 영화를 고를 수 있도록 화면 구성과 사용 편의성에 중점을 두었습니다.
